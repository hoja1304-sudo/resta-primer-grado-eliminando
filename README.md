# RestaPrimerGradoEliminando

Juego web de resta eliminando objetos para primer grado.

## Uso

Abre `index.html` en un navegador. No necesita servidor ni dependencias.

## Familias

- De 1 a 3
- De 1 a 5
- De 1 a 19

El estudiante toca objetos para eliminar la cantidad del sustraendo y luego selecciona el resultado en burbujas.
